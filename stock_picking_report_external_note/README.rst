This module extends the functionality of stock picking to support external notes
and to allow you to display them in the external reports.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_

Usage
=====

To use this module, you need to:

#. Go to stock picking
#. Fill *External notes* field on the Notes tab
#. Print the Delivery Slip report and check that *External notes* has been printed
